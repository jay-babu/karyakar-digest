2020 MUKHPATH ASSESSMENT
Mukhpath is essential to satsang growth and is something we place great importance on in sabha. Although we cannot meet in-person, mukhpath can always be practiced at home. For that reason, an at-home mukhpath assessment package is now available.
Mukhpath taught since last year’s exam up to mid-March this year is compiled into a study guide for balaks/balikas to use. Along with the study guide, there are self assessment tools (practice exam, answer sheet, and answer key) included in the package to help balaks/balikas test their knowledge. Parents should actively help their balaks/balikas review and study the mukhpath, then administer the exam. No exams or scores will be collected. Further details can be found in the ReadMe.

BKMS Sabha Reporting Changes
In order to make reporting easier for Group 3 Sabha Goshthi/Hangout we will update Bal/Balika sabha reports on BKMS to allow center’s to select which groups held Sabha Goshthi/Hangout.
Before: In order to record Group 3 Sabha Goshthi/Hangout, the Coordinator/Reporting Karyakar had to select each non Group 3 Balaks and mark them as absent. This was a long and tedious process.
Now: Each Group has a radio button next to it, asking ‘Was Group Sabha/Hangout held?'. The default option will be set to ‘Yes’ for all groups. If the sanchalak answer ‘Yes’, then they must fill attendance for that particular group. However, if the sanchalak marks that question as ‘No’, then all the Balaks/Balikas of that group will automatically be disabled and attendance will not be required for that group.
Note: Hangouts for Group 0, 1, 2 are not required at this time, and in the case where the hangout did not take place, we request sanchalaks to update the weekly sabha report by selecting ‘No’ for their respective group. Keep in mind, this year's attendance will not impact your center's overall status. If any hangouts for these groups do happen, then we kindly request Sanchalaks to input the attendance as per normal.
This modification went into effect on Sunday, June 21st.
